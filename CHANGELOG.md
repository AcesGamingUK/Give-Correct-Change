# Changelog

## 1.0.0

### Initial Release

- Added **Manual Fill** mode.
  - Press the configured hotkey, default **F**, to prepare the exact change.
  - Click the **cash pile** on the counter to hand the change to the customer.
- Added **Auto Fill** mode.
  - Automatically prepares the exact change.
  - Click the **cash pile** on the counter to complete the transaction.
- Added **Auto Complete** mode.
  - Automatically completes cash transactions instantly.
- Added configurable hotkey.
- Added lightweight in-game status panel.
- Added draggable panel with saved position.
- Added ConfigManager-compatible configuration.
- Added startup logging banner.
- Fixed cash calculation issues.
- Fixed Auto Fill money-loss issue.
